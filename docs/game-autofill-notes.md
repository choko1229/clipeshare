# ゲーム名自動補助メモ

投稿メディアからゲーム名を完全自動判定する標準的なメタデータは基本的にありません。
そのため、ゲーム名を空欄にした場合に確実な自動読み取りを行うことは現実的ではありません。

## 現実的な候補

- ファイル名から候補を作る
  - 例: `valorant-ace.mp4` から `valorant` を候補にする
- 投稿者の過去によく使うゲームを候補にする
- 既存 `Game` テーブルから入力候補を出す
- IGDB APIで検索候補を出す

## 推奨

MVPではゲーム名は必須のままにし、入力欄に候補表示を追加する。
空欄投稿を許可する場合は、`Unknown Game` のような仮ゲームに紐づけて、管理画面で後から統合する。
